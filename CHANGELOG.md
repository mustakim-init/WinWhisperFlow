# Changelog

## [0.1.0] — 2025-06-29

### Added
- Initial release of WinWhisper Flow
- Real-time voice transcription with `faster-whisper` (CPU) and `sherpa-onnx` (GPU)
- File transcription with progress tracking and cancel support
- Music transcription with Demucs source separation
- Phone mic streaming via local HTTP server with QR code pairing
- Model download management (download, pause, resume, cancel)
- Hardware info page (CPU, RAM, GPU detection)
- Editable transcripts with copy support
- Dark mode with system preference detection
- Full auto-setup flow for first-time users
